## Getting Started

### Prerequisites
```sh
node
```
```sh
npm
```

### Installation

1. Clone the repo
```sh
git clone https://github.com/villivald/proj_react_2020.git
```

1. Goto the proj_react_2020
```sh
cd proj_react_2020
```
1. Run these commands.
```sh
npm start
```
